# Changelog

## 0.1.0-alpha.1
- Added activation gate (1-5) with quick vs full check-in flow.

- Introduced overlay hamburger menu and streamlined navigation.
- Standardized layout spacing with tokens and shared containers.
- Added belief valence toggle with new positive beliefs.
- Updated sensations and body areas (added burning/neck; removed pressure/cold/arms_hands/legs).
